Question-1: What are the different ways to select an element in the DOM?

JavaScript provides five ways to select elements: getElementById (by ID), getElementsByClassName (by class), getElementsByTagName (by tag), querySelector (first matching CSS selector), and querySelectorAll (all matching CSS selectors).

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML gets the HTML inside an element, innerText gets only the visible text, and textContent works like innerText but is often a better choice for performance and consistency.

Question-3: What is event delegation in the DOM?

Event delegation means attaching an event to a parent element instead of individual child elements, allowing it to handle events triggered by its children.

Question-5: How do you create, add, and remove elements using JavaScript?

createElement lets us create a new element, appendChild adds it to the page, and removeChild removes an existing element.
